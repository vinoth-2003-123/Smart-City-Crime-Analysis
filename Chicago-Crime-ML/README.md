# Smart City Crime Analysis using Machine Learning

## Project Overview
This project analyzes Chicago crime data using supervised and unsupervised learning techniques.

## Supervised Learning
- Predict arrest likelihood
- Algorithms: Random Forest, XGBoost

## Unsupervised Learning
- Crime hotspot detection
- Algorithms: K-Means, DBSCAN

## Technologies Used
- Python
- Scikit-learn
- XGBoost
- Matplotlib

## Dataset
Chicago Crime Dataset (City of Chicago Open Data)
